📸 这是什么鬼？
这是一个用20张照片就能在人群中找到你朋友的AI项目！
对，你没看错，20张，不是2000张，也不是20000张。
原理？就是让AI死死记住你朋友的脸，然后像警察认嫌疑人一样，在照片里圈出他/她。

🎯 它能干啥？
给你一张合照，它能圈出你朋友（如果它没偷懒的话）

准确率？大概... 看它心情 😅

偶尔会把路人甲当成你朋友，别怪它，它只是个孩子

🚀 怎么用？
第1步：准备照片
在 dataset 文件夹里创建两个小文件夹：

true/：把你朋友的20张照片扔进去（越多越准，但我们就20张，挑战极限！）

false/：把路人甲乙丙丁的照片扔进去（越多越好，让AI知道世界上还有其他人）

注意最好都是正方形，因为我们微调resize的时候把他统一变成224*224像素！
如果你不听劝，那么AI认识的可能是朋友_马脸版或者朋友_良子版，后果自负嘻嘻

第2步：训练AI（让它记住你朋友）
bash
python train.py
然后你就可以去泡杯咖啡，回来就看到：

text
Epoch 1/10 - Train Loss: 0.6979, Acc: 0.5484
Validation Loss: 0.1853, Acc: 0.8750
看到 Acc: 0.8750 了吗？那是AI在说：“我已经记住你朋友的87.5%了！”

第3步：开始找人！
bash
python predict.py --image 你的合照.jpg
然后AI会在照片里圈出它觉得像你朋友的人，并写上：

text
CYT 0.92
0.92？那是AI在说：“我有92%的把握，这就是你朋友！（我的朋友叫CYT，你可以改）”

Q：为什么AI把一只狗圈出来了？

Q：为什么AI什么都没圈出来？

Q：20张照片真的够吗？
